# Create boxes on mc-remote
## 箱を作るAPI
***
このAPIでは、縦、横、高さ、角度などを入力すると自由自在に箱を生成できます。

また、材質の変更、回転、移動、変形などもできます。アニメーションのようにMinecraft内で動きます。

---

# 今後の予定
1. 縦、横、高さ、起点座標、使用するブロックを入力しその通りに箱を生成できるようにする。
2. 設定項目に角度を追加し、X、Y、Z方向どの向きにも角度を変えられるようにする。
3. ブロックの種類を何個も追加できるようにし、指定した間隔で変化するようにする。
4. 設定項目に回転の有無を追加し、X、Y、Z方向どの向きにも指定した速度で回転できるようにする。
5. 移動の設定を追加し、設定した向きに指定した速度で指定した距離移動できるようにする。
6. 変形できるようにする(どのように変形するかは未定)。
